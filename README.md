## 🏢 Project Overview

Donate Today is a modern and responsive donation landing page designed to inspire people to help those in need.  
The website features clean layouts, responsive sections, and an engaging user interface built with Tailwind CSS.

This project focuses on responsive design, reusable utility classes, and modern frontend styling using Tailwind CSS.

---

## ✨ Key Features

- Fully Responsive Design
- Modern Donation Landing Page
- Hero Banner Section
- Donation Amount Cards
- Donation Plan Section
- Inspirational Content Areas
- Clean & Minimal UI
- Built with Tailwind CSS
